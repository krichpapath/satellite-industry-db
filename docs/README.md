# Project Documents

This folder keeps non-runtime materials out of the Next.js deploy root.

- `reference/` - research documents, extracted text copies, and uploaded source references.
- `prototypes/` - earlier static HTML and JSX visual prototypes.
- `extracted/` - unpacked DOCX internals and the extraction helper.
- `archive/` - obsolete scratch work kept for traceability.

The active app lives in `app/`, `components/`, and `lib/`. The AWS API package lives in `backend-lambda/`, and database schema/migration files live in `database/`.
